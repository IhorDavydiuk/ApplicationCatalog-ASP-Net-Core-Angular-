В меню "Tools" -> "Nuget Package Manager" -> "Package Manager Console" ввести команду которая сгенерирует БД: Update-Database
Тогда в папке текущего пользователя на диске С (C:\Users\[Имя_пользователя]) мы сможем найти два файла applicationDb.mdf и applicationDb.ldf, которые и представляют базу данных.
